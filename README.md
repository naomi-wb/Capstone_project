# Capstone Project
This repository includes information needed to understand and reproduce a reanalysis of the paper *Global abundance estimates for 9700 bird species* by Callaghan et al. (2021), conducted as part of the Capstone project at the University of Freiburg. 

# Read me
The paper *Global abundance estimates for 9700 bird species* by Callaghan et al. (2021) can be found as a PDF in the above data structure. Additionally, the supplimentary information and read me provided by the authors for the study are also included.

Within the document "Re-analysis_report", it is possible to find the reanalysis conducted, and in file "Comment" the comment to the authors. Within the "R_scripts" folder, all scripts needed to reproduce the paper are present, along with those needed for the modifications we tried. The figures folder houses all figures we created. Please note that the code for these is only partially available.
